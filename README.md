# FinTrack — Finance & Inventory App

A full-stack web app built with Flask, MongoDB, and Bootstrap.

## Features
- User authentication (register/login/logout)
- Expense Tracker with income/expense tracking and charts
- Inventory Management System with stock alerts and history
- Dashboard with Chart.js visualizations

## Tech Stack
- Backend: Flask + PyMongo
- Database: MongoDB Atlas
- Frontend: Bootstrap 5 + Chart.js
- Deployment: Render.com
